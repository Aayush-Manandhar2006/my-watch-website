# My Watch Website 🕐

A static e-commerce style website for watches, hosted on AWS S3.

## Live Site
http://aura-watches.s3-website-us-east-1.amazonaws.com/

## Tech Stack
- HTML, CSS, JavaScript
- AWS S3 (Static Website Hosting)
- Git & GitHub (Version Control)

## Project Structure
- `Css/` — Stylesheets
- `Images/` — Watch images (Ap, Pp, Rm collections)
- `Js/` — JavaScript files
- `Pages/` — Additional HTML pages
- `Videos/` — Video assets

## ☁️ Deployment
This website is deployed using AWS S3 static website hosting
via an AWS Academy Sandbox environment.
